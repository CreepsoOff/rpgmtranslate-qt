# Batch Processing

RPGMTranslate allows to easily apply some batch processing to the translation.

## Batch Menu

Batch menu consists of batch action select, translation column select and file select menu.

## Batch Action

There are three batch actions:

1. Trim - Removes leading and trailing whitespace from translation text.
2. Translate - Translates source text and fills **all** the translation fields. If you don't want your translation overwritten, create a dedicated column for the machine transaltion.
3. Wrap - Wraps translation text to the specified limit. This is the most dangerous tool, since it can make your translation display incorrectly, so **use with caution**.
