# Asset Inspector

Built-in asset inspector allows to browse through the assets of the opened projects and inspect them.

## Image Viewer

Encrypted (`.png_`, `.rpgmvp`) and decrypted (`.png`, `.jpg`) images are supported.

Viewer allows you to drag the viewport and zoom in/out.

## Code Viewer

Code viewer allows you to comfortably view JSON, JavaScript and Ruby code.

### Syntax Highlighting

For JSON, JavaScript and Ruby files there's built-in syntax highlighting. It just works out of the box.

### Beautify

For JSON, there's a built-in beautification of the contents. It will format JSON contents for easy readability.

It's planned to also bring JavaScript and Ruby code formatting.

## Media Player

For video and audio files, there's a built-in media player.

It allows to play audio and video assets, both encrypted (`.ogg_`, `.rpgmvo`, `.m4a_`, `.rpgmvm`) and decrypted (`.ogg`, `.m4a`, `.mp4`, `.webm`).

## Font viewer

Font viewer simply shows some sample text, along with supported writing systems. Writing systems are displayed so that you know whether your language is supported by the font.
